
# LIDDY UI

This project was created to help you create a UI from the Quasar framework. The Quasar framework was created using Vue JS which is very powerful for creating spa/PWA apps.



## How To Use

To start project

Clone this repository

```bash
  npm install
```
and then

```bash
  quasar dev
```
